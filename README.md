Getting started
---------------
documentation on how to setup and run the project

Usage
-----
Clone the repository and run the following commands:

```bash
$ git clone git@github.com:makwasi1/compulynx-api.git .
$ cd compulynx-api
$ mvn clean install
$ mvn spring-boot:run
check the application on http://localhost:8080
```
